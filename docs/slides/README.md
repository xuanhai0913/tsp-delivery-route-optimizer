# Slides

Suggested slide structure:

1. Problem statement: shortest path on a weighted map graph.
2. Dataset model: nodes, weighted edges, source, target.
3. Dijkstra idea and complexity.
4. A* idea, `f(n) = g(n) + h(n)`, and heuristic notes.
5. Bellman-Ford repeated relaxation and negative-cycle detection.
6. Floyd-Warshall dynamic programming matrix for all-pairs shortest paths.
7. API and UI architecture.
8. Experiment comparison: path cost, runtime, visited nodes.
9. Demo script and screenshots.
10. Academic references from `docs/references.md`.
11. Lessons learned and future backend solver work.
